Seguridad con Números Primos

Problema Cotidiano

Actualmente muchas personas utilizan contraseñas débiles y fáciles de adivinar, poniendo en riesgo su información personal.

Los números primos son fundamentales en la seguridad informática y la criptografía, ya que ayudan a generar claves más difíciles de descifrar.

¿Qué es un número primo?

Un número primo es aquel que solamente puede dividirse entre 1 y sí mismo.

Ejemplos: 2, 3, 5, 7, 11, 13, 17...

¿Por qué sirven en seguridad?

Los sistemas de seguridad modernos utilizan números primos para crear algoritmos de cifrado y proteger información bancaria, contraseñas y datos privados.

Mientras más complejo sea el número, más difícil será romper la seguridad.

Explicación del Algoritmo

El algoritmo utilizado en este proyecto analiza si un número es primo mediante un ciclo for.

El sistema divide el número ingresado entre todos los valores desde 1 hasta el mismo número.

Si el número solamente tiene dos divisores exactos:

  • El número 1
  • Y el mismo número

Entonces el sistema determina que el número es primo.

En cambio, si el número tiene más divisores, significa que no es primo y no sería tan seguro para representar una clave básica de seguridad.

Este algoritmo utiliza:

  • Variables let
  • Ciclos for
  • Condicionales if
  • Manipulación del DOM con getElementById

Gracias a esto, la página puede analizar números dinámicamente y mostrar resultados en tiempo real.
