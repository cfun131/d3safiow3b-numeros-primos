# Seguridad con Números Primos

## Descripción del Proyecto

Este proyecto consiste en una página web interactiva que analiza si un número puede representar una clave básica de seguridad utilizando números primos.

El sistema permite ingresar un número y verificar automáticamente si es primo mediante un algoritmo desarrollado en JavaScript.

Además, la página explica cómo los números primos son utilizados en la seguridad informática y la criptografía para proteger información digital.

---

## Problema Real

Actualmente muchas personas utilizan contraseñas débiles y fáciles de adivinar, poniendo en riesgo información personal, cuentas digitales y datos privados.

Los sistemas modernos de seguridad utilizan números primos debido a que son difíciles de factorizar matemáticamente, permitiendo crear algoritmos de cifrado más seguros.

Este proyecto busca representar de manera educativa cómo los números primos pueden relacionarse con la ciberseguridad y la protección de datos.

---

## Algoritmo Utilizado

El algoritmo verifica si un número es primo utilizando un ciclo `for`.

El sistema divide el número ingresado entre todos los valores desde 1 hasta el mismo número.

Si el número solamente tiene dos divisores exactos:

* 1
* el mismo número

entonces el sistema determina que el número es primo.

En cambio, si el número tiene más divisores, significa que no es primo y no sería una buena opción para representar una clave segura.
